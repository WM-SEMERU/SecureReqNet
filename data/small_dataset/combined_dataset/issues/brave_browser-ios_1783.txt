No option to cancel sync chain creation when device is in airplane mode


Description:
No option to cancel sync chain creation when device is in airplane mode
Steps to Reproduce

Put device on airplane mode
Enable sync from setting
Goes into an infinite loop. No option to go back or cancel sync chain creation. Only option is to remove app from memory

Actual result:
No option to cancel sync chain creation when device is in airplane mode. Sync chain created when app is relaunched after removing from memory
Expected result:
Sync chain creation should fail when device is in airplane mode. No sync chain should be created after app is relaunched. Sync chain should only be created when device in online
