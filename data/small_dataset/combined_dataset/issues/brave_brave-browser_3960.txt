Cannot open links in mail.com emails


Description
Brave users with mail.com accounts report that opening links from within emails in Brave does not work as intended. Reports claim that when clicked an error code 403 with “Oops, something went wrong” is displayed.
Steps to Reproduce

(mail.com account required) Visit mail.com and login
Open an email containing hyperlink to external site
Click hyperlink

Actual result:
403 error is thrown:

Expected result:
Link should open in a new tab
