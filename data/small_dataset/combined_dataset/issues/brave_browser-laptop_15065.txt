Editing a bookmark url creates a duplicate bookmark


Description
Editing a bookmark duplicates the bookmark on other synced devices.
Steps to Reproduce

Two browsers connected via sync and have the same bookmarks in the toolbar
In the first computer change bookmark url a bit like cleaning it up
In the second computer the bookmark remains unchanged and a new bookmark is added with the new url at the end of the bookmark list

Actual result:
Duplicate bookmark with new url
Expected result:
Bookmark url change without a duplicate
Reproduces how often:
100%
