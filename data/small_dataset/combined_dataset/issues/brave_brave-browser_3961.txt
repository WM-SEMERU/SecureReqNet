Style Chrome browser dialog with the Brave themed dialog component


Description
Currently, our browser dialogs follow the default Chrome theme. We need to style them with our Brave themed dialog component and de-Chromify our browser.
Designs
The Brave themed dialog component interaction guidelines and CSS can be found in Abstract: https://share.goabstract.com/d07d689f-a53e-480d-941c-6afed1d916b9. Use the "Inspect" tab to see layout specs.
