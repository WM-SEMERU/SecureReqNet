File menu pops up issue


Description
Issue is reported by community.
https://community.brave.com/t/navigation-issue-menu-pops-out/32813
When user enters any website in the URL bar, after hitting ctrl+enter a File menu pops up automatically.
Steps to Reproduce

Launch Brave
Press Alt+D shortcut key.
Verify that the focus goes into the address bar, also File menu options pops up
Enter a website e.g: cnet.com in the address bar
Click on Ctrl+Enter

Actual result:
Upon clicking Ctrl+Enter, the File menu pops up automatically

Expected result:
File menu should not display
Reproduces how often:
Always
